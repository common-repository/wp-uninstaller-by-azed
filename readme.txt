This plugin allow you to delete your wordpress installation safely.

== Description ==

You need to uninstall your wordpress site ?

But don't want to do it manually ?

Need guidelines ?

* It show you tables in your database starting with the prefixe setted in your wp-config
* You can unsellect tables you don't want to delete
* You'll need to confirm before it delete your wordpress

**ONLY ADMINISTRATOR CAN USE IT !**

* Available in ENGLISH
* Available in FRENCH
* .PO available to make your own translation !


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-uninstaller-by-azed` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. It's ready !
